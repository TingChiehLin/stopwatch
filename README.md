# StopWatch

This project was aimed to make a StopWatch with Seven Segment Display

## Technical Use & Resoucres

• React

• ES6

• Tailwind CSS

• Redux Toolkit

### `npm start`

Runs the app in the development mode.\
Open [http://localhost:3000](http://localhost:3000) to view it in your browser.
